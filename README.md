# 机兵防卫圈 — 崩坏战

[![CI](https://github.com/tnpp712/battle-game/actions/workflows/ci.yml/badge.svg)](https://github.com/tnpp712/battle-game/actions/workflows/ci.yml)
![Python](https://img.shields.io/badge/Python-3.9%2B-3776AB?logo=python&logoColor=white)
![Pygame](https://img.shields.io/badge/Pygame-2.5%2B-2ea44f)
![Platform](https://img.shields.io/badge/platform-macOS%20%7C%20Linux%20%7C%20Windows-555)

模仿《十三机兵防卫圈》战斗（崩坏战）模式的塔防 / 即时策略小游戏，去掉剧情。
你指挥一支机兵小队守护中央**终端**，抵御一波波从四面来袭的机械兽，并可在战场布防、升级基地。

> 纯 Python + Pygame，**无任何外部素材**——所有单位与特效均为矢量绘制（抗锯齿 + 辉光 + 2× 超采样）。

## 下载即玩（推荐）

到 [**Releases**](https://github.com/tnpp712/battle-game/releases) 下载对应平台的包，双击运行，无需安装 Python：

- **macOS**：下载 `BattleGame-macOS.zip` → 解压 → 双击 `BattleGame.app`
- **Windows**：下载 `BattleGame.exe` → 双击

> 应用未做代码签名，**首次打开需手动放行**：
> - macOS：右键 `BattleGame.app` →「打开」→ 弹窗里再点「打开」（或 `xattr -dr com.apple.quarantine BattleGame.app`）
> - Windows：若弹 SmartScreen，点「更多信息」→「仍要运行」

## 从源码运行（开发者）

```bash
pip install -r requirements.txt
python3 main.py
```

需要 Python 3.9+ 与 Pygame。

无窗口冒烟测试（CI 亦运行此脚本）：

## 玩法

游戏在**建造阶段**与**战斗阶段**之间循环，共 6 波。

### 核心机制：实时 + 主动暂停
战斗实时进行。随时按 **空格** 暂停，在暂停下从容地给机兵下达移动 / 攻击指令，
再按空格继续 —— 还原十三机兵的指挥手感。

### 机兵操作
- **左键点击机兵** → 选中（**Shift+点击**加/减选；**空地拖拽框选**多个）
- **从选中机兵身上拖拽** → 画移动路径（多选时为编队移动到终点）；**空地单击**=移动到点
  - 路径**不能穿过障碍**（地形 / 自己造的墙）。可行=绿色，违规=红色，自动尝试绕行
- **右键点击敌人** → 整组指定攻击目标；**右键空地** → 整组停止
- **Ctrl+1~6** 设编组，**1~6**（战斗中）选组
- **Q / E** → 释放选中机兵的两个技能（消耗能量 + 冷却）。点位型技能按键后进入瞄准态，**左键确认、右键/ESC 取消**。HUD 技能槽也可直接点击
- **M** → 静音 / 取消静音

每个机兵两个技能（能量战斗中回复、攻击命中额外回能）：
| 机兵 | 定位 | Q | E |
|------|------|------|------|
| 近接 | 肉盾近战 | 震荡冲击（范围爆发+眩晕）| 战术突进（位移+落点伤害）|
| 炮击 | 远程 AOE | 饱和轰炸（点位大范围）| 校准模式（限时增伤增程，生根）|
| 突击 | 高速点射 | 弹幕扫射（射程内多段）| 过载冲锋（限时攻速移速）|
| 支援 | 治疗 | 过载治疗（全队回血）| 力场护盾（点位友军护盾）|
| 电磁 | 控制 | 静电脉冲（范围眩晕）| 磁轨钉（点位减速力场）|
| 工程 | 召唤 | 部署哨戒炮（点位自动炮台）| 维修无人机（点位治疗机）|

### 终端全局技能（消耗资源，左上角技能栏）
- **Z 轨道炮** — 点位巨额范围伤害
- **X 全场过载** — 全体机兵限时增伤 + 攻速 + 移速
- **C 时滞力场** — 全场敌人限时减速

### 建造工事（资源消耗）
在 HUD 建造栏点击，或用数字键：
- **1 阻挡墙** — 障碍物，挡住地面敌人的去路（飞行单位无视），可被攻击
- **2 自动炮塔** — 射程内自动开火
- 选中墙/炮塔后在战场上点击放置（绿色=可放，红色=不可）

### 升级基地（点击即升，可重复）
- **3 维修基站** — 治疗终端附近的机兵
- **4 护盾发生器** — 提升终端血量上限并持续回复
- **5 雷达阵列** — 提升所有炮塔射程

### 资源
击杀敌人、清除每波都获得资源，用于建造与升级。

### 流程键
- **N** — 建造阶段开始下一波
- **空格** — 战斗中暂停 / 继续
- **ESC** — 取消选择 / 取消建造
- **R** — 重新开始
- **Shift+ESC** — 退出

## 胜负
- 守住全部 6 波 → 防卫成功
- 终端血量归零 → 失败

## 文件结构
- `config.py` — 所有数值与平衡配置（机兵 / 敌人 / 建造 / 波次）
- `geometry.py` — 几何计算（路径阻挡检测、避障转向、碰撞）
- `entities.py` — 实体（终端 / 机兵 / 敌人 / 炮塔 / 墙 / 特效）
- `ui.py` — HUD 与状态绘制
- `game.py` — 状态机、事件、更新与渲染编排
- `main.py` — 入口与主循环

想调难度/数值，直接改 `config.py` 即可。
